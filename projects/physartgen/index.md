## Overview

PhysArtGen is a related research project on generating articulated 3D assets for embodied simulation. The project connects text understanding, part-level asset generation, and URDF construction so that generated objects can support robot interaction in simulation.

My contribution focuses on physical-consistency design, URDF optimization, dataset construction, and experimental evaluation. This page is intended as a concise project showcase rather than a publication page.
