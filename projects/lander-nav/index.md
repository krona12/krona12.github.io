## Overview

LANDER-Nav is a research project on open-vocabulary embodied navigation. The main idea is to keep the structure of a language goal instead of compressing the whole instruction into a single query. The agent reasons with target, anchor, context, and relation cues, so it can better distinguish similar objects and choose a more reliable stopping viewpoint.

This page is only a brief project showcase.
